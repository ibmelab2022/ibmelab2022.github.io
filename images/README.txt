이 폴더에 사진을 올리세요.

[사진 올리는 법]
1) GitHub 저장소에서 이 images 폴더를 클릭
2) 오른쪽 위 "Add file" → "Upload files" 클릭
3) 사진 파일을 끌어다 놓고 (예: prof.jpg, member1.jpg 등)
4) 아래 초록색 "Commit changes" 버튼 클릭

[사진 쓰는 법]
data.js 파일에서 photo 칸에 아래처럼 적으면 됩니다.
   photo: "images/prof.jpg",

권장: 파일 이름은 영어+숫자로 (한글 파일명은 가끔 문제가 생깁니다).
권장 크기: 인물 사진은 가로 600px 내외면 충분합니다.
