# IBME Lab 홈페이지

강남대학교 IBME Lab. 연구실 홈페이지입니다.
**코딩을 몰라도** `data.js` 파일 하나만 고치면 내용을 바꿀 수 있게 만들었습니다.

---

## 📁 파일 구성 (3개뿐입니다)

| 파일 | 역할 | 교수님이 만질 일 |
| --- | --- | --- |
| **`data.js`** | 홈페이지에 들어가는 **모든 내용**(소개·멤버·논문·소식 등) | ✅ **이것만 고치면 됩니다** |
| `index.html` | 홈페이지의 디자인/틀 | ❌ 건드릴 필요 없음 |
| `images/` | 사진을 올려두는 폴더 | 사진 넣을 때만 |

---

## 🚀 처음 1번만: 홈페이지 켜기 (약 5분)

> 아래는 완전 무료입니다. 카드 등록 없음, 광고 없음.

### 1단계. GitHub 계정 만들기
- https://github.com 접속 → **Sign up** → 이메일로 가입 (무료)

### 2단계. 새 저장소(repository) 만들기
1. 오른쪽 위 **`+`** → **New repository** 클릭
2. **Repository name** 칸에 이렇게 입력 → **`아이디.github.io`**
   - 예: 아이디가 `jihunkim` 이면 → `jihunkim.github.io`
   - ⚠️ 이 이름 규칙을 지켜야 주소가 깔끔하게 나옵니다.
3. **Public** 선택 (무료 홈페이지는 Public 이어야 합니다)
4. 아래 **Create repository** 클릭

### 3단계. 파일 3개 올리기
1. 방금 만든 저장소 화면에서 **Add file** → **Upload files** 클릭
2. 이 폴더 안의 **`index.html`**, **`data.js`** 와 **`images` 폴더**를 통째로 끌어다 놓기
3. 아래 초록색 **Commit changes** 클릭

### 4단계. 홈페이지 스위치 켜기 (Pages 활성화)
1. 저장소 위쪽 **Settings**(⚙️) 클릭
2. 왼쪽 메뉴에서 **Pages** 클릭
3. **Branch** 를 `None` → **`main`** 으로 바꾸고 **Save**
4. 1~2분 기다리면 이 주소로 홈페이지가 뜹니다 👉 **`https://아이디.github.io`**

끝났습니다! 🎉

## ✏️ 앞으로 내용 바꾸기 (매번 이것만)

새 논문이 나왔거나, 멤버가 들어왔거나, 소식을 올리고 싶을 때:

1. GitHub 저장소에서 **`data.js`** 파일을 클릭
2. 오른쪽 위 **연필(✏️ Edit) 아이콘** 클릭
3. 내용 수정 (아래 "자주 하는 편집" 참고)
4. 맨 아래 **Commit changes** 클릭
5. **1~2분 뒤 홈페이지에 자동 반영** ✅

> 💡 워드에서 글자 고치는 것과 똑같습니다. 겁내지 마세요.

## 📝 자주 하는 편집 3가지

`data.js` 안에는 자세한 한글 설명이 각 항목마다 붙어 있습니다. 대표 3가지만 미리 보여드리면:

### ① 새 논문 추가
`publications` → `journalIntl` 목록 **맨 위**에 한 줄 추가:
```js
{ c: "저자, (2026). 논문 제목. Journal Name, 12(3).", doi: "10.1109/example" },
```
- `doi`(논문 고유번호)를 넣으면 **DOI 링크 배지**가 자동으로 붙습니다.
- `doi`가 없으면 → `{ c: "저자, (2026). 논문 제목. Journal Name." },` 처럼 doi 부분만 빼면 됩니다.

### ② 새 멤버 추가
`students` 목록에 `{ }` 한 덩어리 복사해서 추가:
```js
{
  name:  "이순신 (Sunsin Lee)",
  role:  "M.S. Student",
  email: "sslee@kangnam.ac.kr",
  photo: "images/sslee.jpg",   // 사진 없으면  photo: ""  (이름 이니셜이 예쁘게 표시됨)
},
```

### ③ 새 소식(공지) 올리기
`news` 목록 **맨 위**에 추가:
```js
{
  date: "2026.06.15",
  text: `2026 IEEE IUS 학회에서 포스터 발표를 진행했습니다. **우수상**을 수상했습니다!`,
  images: [],   // 사진 있으면  images: ["images/ius2026.jpg"]
},
```
- 글 안에서 `**별표 두 개**` 로 감싸면 **굵게** 표시됩니다.
- 줄을 바꾸고 싶으면 그냥 엔터를 치면 됩니다.

## 🖼️ 사진 넣는 법

1. GitHub 저장소에서 **`images`** 폴더 클릭 → **Add file** → **Upload files**
2. 사진 올리고 **Commit**
3. `data.js`에서 `photo: "images/파일이름.jpg"` 라고 적으면 끝

> 파일 이름은 **영어+숫자**로 지어주세요 (예: `prof.jpg`). 한글 파일명은 가끔 안 나올 수 있습니다.

## ⚠️ 편집할 때 딱 3가지만 조심

1. **따옴표(`"`)와 백틱(`` ` ``)** 은 짝을 맞춰서, 그 **안쪽 글자만** 바꾸세요. 따옴표 자체를 지우면 안 됩니다.
2. 각 줄/덩어리 끝의 **쉼표(`,`)** 는 그대로 두세요.
3. 새 항목은 **기존 것을 통째로 복사한 뒤 내용만** 바꾸는 게 제일 안전합니다.

혹시 실수해서 화면이 이상해지면, GitHub의 **History**에서 이전 버전으로 되돌릴 수 있으니 안심하세요.
