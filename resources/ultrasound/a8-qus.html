부록 · Advanced Ultrasound Techniques · A8

Quantitative Ultrasound정량 초음파 영상

B 모드의 밝기는 숫자가 아닙니다 — 장비·이득·설정을 바꾸면 같은 조직이 다르게 나옵니다. 그런데 02장이 심어둔 "c = 1540 가정"과 17장이 남긴 "스페클은 조직의 통계", 이 둘에서 조직 고유의 숫자를 뽑아낼 수 있습니다.

먼저 — 장비는 음속을 "가정"한다

12·13장의 빔포밍 지연은 전부 c = 1540 m/s 를 넣고 계산합니다. 실제 조직의 c 가 다르면 지연이 틀리고, 틀린 지연은 초점을 무너뜨립니다.

  • 대가 · 개구 가장자리와 가운데의 지연 오차 ≈ (D²/8z)(1/c − 1/c₀). 5 MHz·개구 20mm·초점 40mm 면 ±3.2% 만 벗어나도 위상 오차가 45°를 넘습니다. 지방(1450 m/s)은 −5.8% — 이미 무너집니다. 05장의 "안 보이는데 빔은 휜다"가 이것입니다.
  • 뒤집기 · 그렇다면 가정 음속 c₀ 를 훑어 가장 또렷해지는 값을 찾으면, 그게 진짜 c 입니다. 아티팩트의 원인을 측정 도구로 바꾼 거죠.
  • 왜 쓸모 있나 · 지방은 c 가 낮습니다. 지방 30% → 1520 m/s — 지방간을 숫자로 잴 수 있습니다.
인터랙티브 · 음속을 틀리면 초점이 무너진다 — 그래서 잴 수 있다—
음속 오차—%
지연 오차—ns
위상 오차—°
결맞음—%
판정—

위 곡선은 가정 음속 c₀ 를 훑었을 때의 결맞음입니다 — 봉우리가 실제 c 에 정확히 섭니다. 이게 음속 영상의 원리입니다.
아래는 그때의 측방향 빔. c₀ 를 실제 c 에서 떼어놓아 보세요 — 주엽이 무너지고 사이드로브가 솟습니다. 지방(1450)에 1540을 가정하면 위상 오차 91°입니다.

01

음속 영상Speed-of-sound imaging

실전에서는 화면 전체를 한 값으로 재지 않습니다. 구역마다 c₀ 를 훑어 결맞음이 최대가 되는 값을 찾으면 음속 지도가 됩니다. 개구를 반으로 갈라 두 부분영상의 상호상관을 보는 방법, 각도별 평면파(A1)의 위상 정합을 보는 방법 등이 쓰입니다.

지방fat1450−5.8%91° — 붕괴
물water1480−3.9%59° — 붕괴

02장의 씨앗이 여기서 회수됩니다

02장에서 c = √(K/ρ) 를 배우고 "장비는 1540 을 가정한다"고 했습니다. 그게 속도 아티팩트의 씨앗이었죠. 같은 오차가 지금은 신호입니다 — 아티팩트를 지우려 하지 않고 읽으면 조직의 성질이 나옵니다.

02

후방산란계수Backscatter coefficient

17장은 "스페클은 잡음이 아니라 조직의 통계"라고 했습니다. 그 통계를 읽습니다. 후방산란계수 backscatter coefficient, BSC 는 단위 부피·단위 입체각당 되돌아오는 파워입니다 — 조직 고유의 물리량이죠.

인터랙티브 · 스펙트럼의 기울기가 산란체 크기를 말한다—
ka @6MHz—
스펙트럼 기울기—
왕복 감쇠 @6MHz—dB
보정 안 하면—

청록 = 참(보정된 BSC), 빨강 = 감쇠 보정 전입니다.
산란체가 작으면(ka ≪ 1) 기울기가 4 — 06장의 레일리 산란 그대로입니다. 커지면 ka~1 에서 꺾이며 기울기가 떨어지죠 — 그 기울기가 곧 크기입니다. 그런데 감쇠(7장)도 스펙트럼을 기울입니다 — 먼저 걷어내지 않으면 크기를 잘못 읽습니다.

참조 팬텀 — 장비를 나눠서 지운다

받은 스펙트럼에는 조직 말고도 탐촉자 응답·송신 파형·이득·집속이 전부 섞여 있습니다. 성질을 아는 참조 팬텀을 같은 설정으로 찍어 나누면 그 공통항이 소거됩니다.

남는 건 조직 고유의 BSC — 장비가 바뀌어도 같은 숫자입니다. 이게 "정량"의 뜻이죠. 여기서 유효 산란체 크기와 음향 집중도를 뽑아 지방간·종양 특성화에 씁니다.

03

왜 이게 어려운가Why it is hard

  • 감쇠와 산란이 얽힌다 · 스펙트럼 기울기는 산란체 크기와 감쇠 둘 다에서 옵니다. 깊이 3cm·6MHz면 왕복 감쇠만 18 dB — 먼저 풀어야 크기를 읽습니다.
  • 스페클은 통계라 표본이 필요하다 · BSC 한 점을 얻으려면 여러 분해능 셀을 평균해야 합니다. 그래서 공간 분해능을 희생합니다 — 17장의 스페클이 여기서 대가를 청구하는 셈이죠.
  • 가정이 많다 · 산란체가 무작위·균일·약산란이라고 가정합니다. 실제 조직은 구조가 있어 "유효" 크기일 뿐 진짜 세포 크기가 아닙니다.
그래도 방향은 분명합니다

B 모드의 밝기는 비교할 수 없는 수입니다. QUS 는 비교할 수 있는 수를 냅니다 — 다른 장비, 다른 날, 다른 병원에서도요. 추적 관찰과 다기관 연구가 가능해지는 지점입니다. A9 의 전단파 속도도 같은 정신입니다.

한 장 정리 · Summary
  • B 모드 밝기는 숫자가 아니다 · 장비·이득·설정에 따라 달라짐 → 비교 불가
  • 장비는 c=1540 을 가정(02장) · 틀리면 지연 오차 ≈ (D²/8z)(1/c − 1/c₀)
  • 허용 ±49 m/s = ±3.2% (45° 기준) · 지방 1450 = −5.8% → 위상 91° → 초점 붕괴
  • 뒤집으면 측정 · c₀ 를 훑어 결맞음 최대인 값이 진짜 c → 음속 영상
  • 지방 30% → 1520 m/s · 지방간을 숫자로 (아티팩트를 지우지 말고 읽는다)
  • BSC · 17장의 "스페클 = 조직의 통계" 를 숫자로 · 단위 부피·입체각당 되돌아오는 파워
  • 기울기 = 산란체 크기 · ka≪1 이면 4(레일리, 06장) · ka~1 에서 꺾임
  • 감쇠(7장)를 먼저 보정 · 3cm·6MHz 왕복 18dB — 안 하면 크기를 잘못 읽음
  • 참조 팬텀으로 나눔 · 탐촉자·이득·집속이 소거 → 장비 무관한 조직 고유 값
  • 대가 · 표본이 필요해 공간 분해능 희생 · "유효" 크기일 뿐 · 가정이 많음
