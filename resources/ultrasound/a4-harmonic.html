Harmonic Imaging하모닉 영상

01~08장은 내내 "조직은 선형"이라고 가정했습니다. 아닙니다. 압축부가 희박부보다 빨리 갑니다. 파형이 스스로 일그러지며 없던 주파수를 만들어내죠. 그런데 이 오차가 오히려 더 깨끗한 영상을 줍니다.

먼저 — 하모닉 영상이 무엇인가

보내는 건 f₀ 하나인데, 받는 신호에는 2f₀ 가 섞여 있습니다. 조직이 만들어낸 겁니다. 그 조직이 만든 주파수만 골라 영상을 만드는 게 하모닉 영상입니다.

  • 왜곡 · 압축부가 희박부보다 빨리 갑니다(c = c₀ + βu). 사인이 진행할수록 톱니로 기울죠.
  • 하모닉 생성 · 톱니를 푸리에로 펴면 2f₀·3f₀ 입니다 — 없던 주파수가 전파 도중에 생깁니다.
  • 고대역 필터 high-pass / band-pass · f₀ 를 잘라내고 2f₀ 만 남겨 그것으로 영상을 만듭니다. 조직이 스스로 만든 신호라 오히려 더 깨끗합니다.

왼쪽이 파형, 오른쪽이 스펙트럼입니다. 진행할수록 사인이 톱니로 기울며 2f₀, 3f₀ 가 자랍니다.
음압을 반으로 줄여보세요 — 2차 하모닉은 1/4로 떨어집니다. 하모닉 ∝ p² 입니다. 그래서 세게 쏘는 곳(초점)에서만 생깁니다.

01

음속이 압력에 의존한다The nonlinearity parameter B/A

02장의 c = √(K/ρ) 는 작은 진폭에서만 맞습니다. 진폭이 커지면 매질이 눌린 곳에서 더 단단해지고, 게다가 매질 자체가 입자 속도만큼 실려 갑니다. 둘을 합치면 국소 음속이 이렇게 됩니다:

c = c₀ + β u· β = 1 +B2A
u = 입자 속도 · 비선형 파라미터 B/A · 간 6.8 → β = 4.4 · 지방 10.0 → β = 6.0

압축부(u > 0)는 빨리, 희박부(u < 0)는 느리게 갑니다. 사인파가 앞으로 기울며 톱니가 되고, 톱니는 푸리에로 펴면 2f₀, 3f₀ … 입니다. 없던 주파수가 전파 도중에 생기는 것이죠.

숫자로 보면 아주 작은 효과입니다

1.5 MPa 면 입자 속도가 0.92 m/s, 음속 변화는 4.0 m/s — 1540 의 0.26% 입니다. 그런데 수 cm 를 누적하면 파형이 완전히 일그러집니다. 5 MHz·1.5 MPa 에서 충격이 서는 거리가 18.7 mm — 영상 깊이 한복판입니다.

02

왜 하모닉이 더 깨끗한가Three gifts

하모닉 음원의 세기가 국소 압력의 제곱에 비례한다는 사실 하나에서 세 가지가 한꺼번에 나옵니다.

인터랙티브 · 기본파 빔 vs 하모닉 빔—
기본파 −6dB—mm
하모닉 −6dB—mm
빔 폭 비—배
사이드로브 차—dB

하모닉 음원 ∝ p² 이므로 빔이 기본파의 제곱입니다 — dB 로는 모든 값이 두 배가 되죠.
주엽은 0.73배로 좁아지고, 사이드로브는 −13 dB 에서 −27 dB 로 주저앉습니다. 클러터가 통째로 죽습니다.

  • 주엽이 좁다 · 1.207 λ₀F# → 0.886 λ₀F# (0.73배). 측방향이 좋아집니다.
  • 사이드로브가 죽는다 · −13 dB → −27 dB. 빔 옆에서 새어 들어오는 클러터가 사라집니다.
  • 근거리장에 아예 없다 · 하모닉은 쌓여야 생깁니다(위 c1 의 σ). 그래서 지방층·복벽에서 생기는 수차와 다중반사가 하모닉 대역에는 실리지 않습니다. 비만 환자·어려운 창에서 조직 하모닉 영상 THI 가 극적인 이유입니다.
03

대가What it costs

주파수가 2배 → 감쇠가 2배 (7장)

깊이 5 cm 에서 기본파는 왕복 25.0 dB 를 잃습니다. 하모닉은 갈 때 5 MHz(12.5) + 올 때 10 MHz(25.0) = 37.5 dB — 12.5 dB 더 손해입니다. 하모닉은 깊이를 잃습니다.

  • 대역폭을 크게 먹는다 · f₀ 와 2f₀ 를 둘 다 담으려면 탐촉자 비대역폭이 넓어야 합니다(10장). 좁으면 f₀ 를 낮춰야 하고, 그러면 분해능을 잃습니다.
  • 필터가 겹친다 · f₀ 의 꼬리가 2f₀ 대역으로 새어 들어옵니다 (harmonic leakage). 필터를 좁히면 축방향이 나빠지고, 넓히면 기본파가 섞입니다. 이 딜레마를 푸는 게 A5 입니다.

그래서 THI 는 얕고 어려운 창에서 이깁니다 — 복부·심장. 깊은 곳에서는 감쇠가 이득을 먹어치웁니다.
