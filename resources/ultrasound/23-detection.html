Back-End Processing · 23

검파와 압축Detection & Log Compression

ADC 가 넘긴 숫자를 FPGA 가 받습니다. 채널을 합쳐 주사선을 만드는 DAS 빔포밍(12·13장)이 첫 디지털 작업이고, 그 결과는 아직 반송파가 실린 RF입니다. 여기서 포락선만 벗겨내고(검파), 100 dB 를 화면이 낼 수 있는 계조로 눌러야(압축) 비로소 영상이 됩니다.

01

먼저 DAS — 채널을 주사선으로Beamforming, the first digital op

ADC 는 소자 채널마다 숫자열을 내놓습니다. 이것들에 깊이별 지연을 주고 더하면 한 방향의 주사선 하나가 됩니다 — 지연-합 delay-and-sum, 수신 빔포밍(12·13장). 아날로그에서 못 하고 디지털에서 하는 이유가 이것입니다: 깊이마다 지연을 바꾸는 동적 집속은 숫자라야 가능하니까요.

어디에 DAS 를 거나? 예전에는 RF 표본에 직접 지연-합을 했는데, 반송파 때문에 표본보다 훨씬 미세한 지연이 필요해 고배율 보간이 들었습니다. 요즘 장비는 흔히 먼저 IQ 로 내리고(다음 절) decimation 한 뒤 그 IQ 에 DAS 를 겁니다 — 기저대역에선 지연이 정수-표본 이동(거친) + 위상회전 e−j2πf₀τ(미세)로 갈라져, RF 보간 없이 정밀하면서 데이터율은 ~10배 낮습니다. 순서가 검파보다 앞서기도, 뒤서기도 하는 셈이죠. 이 장은 그렇게 얻은 주사선 하나를 영상 밝기로 바꾸는 단계입니다.

02

검파 — 반송파를 벗긴다Envelope detection

빔포밍 결과는 ~5 MHz 반송파가 진동하는 RF 신호입니다(스펙트럼은 ±f₀ 양쪽 대칭). 화면에 필요한 건 진폭(포락선)뿐이라 반송파를 벗깁니다. 해석신호를 만드는 길이 둘입니다.
IQ 복조 quadrature demodulation = 하향혼합 → LPF → decimation. RF 에 e−j2πf₀t(=cos·sin)를 곱하면 스펙트럼이 f₀ 만큼 내려가 신호 대역이 0 Hz로, 엉뚱한 상(像)은 −2f₀로 갑니다. LPF 가 −2f₀ 상을 지우면 기저대역 복소신호 I + jQ만 남죠 (포락선 = √(I²+Q²)). 이제 대역이 좁으니 표본을 훨씬 성기게 다시 뽑아도 됩니다 — decimation, 데이터율 ~10배 감소. 이 IQ 는 도플러(위상)와 DAS에도 그대로 쓰입니다. 대신 f₀ 를 알아야 합니다.
힐베르트 변환 Hilbert transform — 90° 위상천이로 짝 x_H 를 만들어 x + j·x_H 를 구성하면, 스펙트럼에서 음의 주파수(−f₀)만 지워지고 +f₀ 는 그대로 남습니다(한쪽 스펙트럼 = analytic signal). f₀ 를 몰라도 되고 반송파가 유지돼(통과대역) decimation 은 그 자체로는 안 합니다. 결과 포락선은 둘이 같습니다 — 무엇이 편한가의 선택입니다.

인터랙티브 · IQ vs Hilbert — 시간 파형 + 주파수 스펙트럼—
지금 방식—

위 = 시간 파형, 아래 = 주파수 스펙트럼. IQ면 하향혼합이 신호를 0 Hz(기저대역)로 내리고(−2f₀ 상은 LPF 가 제거), 대역이 좁아져 decimation(성긴 표본) 가능 — I·Q 는 반송파 없는 저주파. Hilbert면 −f₀ 만 지우고 +f₀ 는 유지(한쪽 스펙트럼), x·x_H 에 반송파가 남습니다. 포락선(앰버)은 둘이 똑같죠.

03

로그 압축 — 100 dB 를 계조로Log compression

검파된 에코의 세기는 강한 경계와 약한 조직 사이에 100 dB(10만 배)나 됩니다. 화면은 256 계조뿐이라 그대로는 못 담습니다. 그래서 로그로 눌러(gray ∝ log 진폭) 넓은 범위를 계조에 매핑합니다 — 로그라야 약한 것과 강한 것을 동시에 보여줄 수 있습니다. 표시할 동적범위 창을 좁히면 대비가 오르고(약한 에코 잘림), 넓히면 미세 계조가 다 보이지만 흐릿해집니다 — 장비의 dynamic range 손잡이입니다.

인터랙티브 · 동적범위 창에 따른 B-모드 영상—

왼쪽 = 실제 B-모드 영상(경계·조직·저에코 병변·낭종·심부), 오른쪽 = 압축 곡선과 각 구조의 회색. 창을 좁히면(예: 40 dB) 병변·낭종이 검게 잘려 경계가 또렷해지고, 넓히면(예: 80 dB) 미세 계조까지 보이지만 전체가 회색으로 흐릿해집니다 — 같은 데이터, 다른 dynamic range 손잡이입니다.

한 장 정리 · Summary
  • DAS · 채널에 지연-합 → 주사선 하나(12·13장) · 요즘은 decimated IQ 에 DAS(정수표본+위상회전 e−j2πf₀τ)
  • 빔포밍 결과 = RF · 스펙트럼 ±f₀ 대칭 — 화면엔 진폭(포락선)만 필요
  • IQ 복조 = 하향혼합+LPF+decimation · ×e−j2πf₀t → 0 Hz · LPF 가 −2f₀ 제거 · 대역 좁아 ↓fs(~10배)
  • Hilbert = 해석신호 · −f₀ 만 제거, +f₀(반송파) 유지 = 한쪽 스펙트럼 · f₀ 불필요 · decimation 은 안 함
  • 같은 포락선, 다른 편의 · IQ=저데이터율·도플러 친화 / Hilbert=f₀ 불필요·구현 단순
  • 100 dB → 256 계조 · 로그 압축(gray ∝ log 진폭)이라야 약·강 동시 표시
  • 동적범위 창 · 좁히면 대비↑(약한 에코 잘림), 넓히면 계조↑(흐릿)
