Pulse Inversion and Amplitude Modulation펄스 반전과 진폭 변조

A4 는 필터로 2f₀ 를 잘라냈습니다. 그런데 기본파의 꼬리가 새어 들어오고, 필터를 좁히면 축방향을 잃습니다. 필터를 아예 안 쓰는 길이 있습니다 — 두 번 쏘고 더하면 선형은 저절로 지워집니다.

먼저 — 왜 두 번 쏘는가

A4 는 필터로 2f₀ 를 잘랐습니다. 그런데 f₀ 의 꼬리가 2f₀ 대역으로 새어 들어옵니다 (harmonic leakage). 필터를 좁히면 축방향을 잃고, 넓히면 기본파가 섞이죠.

  • 펄스 반전 PI · p 와 −p 를 쏘고 더합니다. 조직이 선형이면 응답도 부호가 뒤집혀 합이 정확히 0 — 필터 없이 지워집니다.
  • 진폭 변조 AM · 부호 대신 크기를 바꿉니다. 전체 한 번, 반쪽 두 번을 쏴서 r(p) − 2r(p/2) 로 조합하죠.
  • 살아남는 건 비선형 성분뿐입니다. 오른쪽 스펙트럼을 보면 무엇이 지워지고 무엇이 남는지 보입니다.
인터랙티브 · 두 번 쏘고 더하면 선형이 지워진다
선형 잔류—
2f₀ (짝수)—
홀수 (f₀)—
송신 횟수—회

① 송신 파형 → ② 돌아온 두 에코 → ③ 둘을 조합한 잔류, 세 칸으로 봅니다. 파랑·빨강이 두 에코인데, PI 는 서로 뒤집혀 더하면, AM 은 거의 같아 빼면 — 어느 쪽이든 선형은 정확히 0, 비선형만 남습니다(③은 작아서 ×N 확대). 오른쪽 스펙트럼이 그 결과이고, 판독값도 스펙트럼 성분입니다.
버튼으로 AM 으로 바꿔보세요. PI 는 짝수(2f₀)만 — 넓은 대역 필요. AM 은 홀수도 남기는데 기본파 대역 안(f₀·3f₀)이라 대역폭 요구가 없습니다. a₃ 를 키우면 AM 은 f₀·3f₀ 잔류가 커지지만 PI 는 꿈쩍 안 합니다 — PI 가 홀수를 죽인다는 증거입니다.

01

다항식 하나로 다 보인다A polynomial says it all

산란체의 응답을 압력의 다항식으로 씁니다 — 선형 항 + 2차 + 3차:

r(p) = a₁p + a₂p² + a₃p³
a₁ = 선형(조직 대부분) · a₂ = 짝수 하모닉의 씨앗 · a₃ = 홀수

펄스 반전 pulse inversion, PI 은 p 와 −p 를 쏘고 더합니다. 대입만 하면 끝입니다:

r(p) + r(−p) = 2a₂p²
선형도 3차도 부호가 뒤집혀 사라지고, 2차만 2배로 남습니다 — 주파수로 풀면 2f₀ 성분(진폭 a₂), 단일 에코의 2f₀(a₂/2) 대비 2배
PI 는 홀수 하모닉도 같이 죽입니다

홀수 항은 (−p)³ = −p³ 라 선형과 함께 지워집니다. 살아남는 건 짝수뿐 — 곧 2f₀ 입니다. A4 의 필터가 하던 일을 필터 없이 하는 셈이라, 기본파 누출 harmonic leakage 이 원리적으로 0 입니다. 게다가 축방향을 위해 대역을 좁힐 필요가 없습니다.

02

진폭 변조 — 홀수를 살린다Amplitude modulation

진폭 변조 amplitude modulation, AM 은 부호가 아니라 크기를 바꿉니다 — 반쪽 진폭으로 두 번, 전체 진폭으로 한 번 쏘고 이렇게 조합합니다:

r(p) − 2·r(p/2) = a₂p²/2 + 3a₃p³/4
선형은 지워지고 2차와 3차가 모두 남습니다 — 주파수로는 2f₀(a₂/4) 와 홀수 f₀·3f₀ (수치 확인: 0.1050 ↔ 이론 0.1050)
왜 홀수를 살리는 게 중요한가 — 대역폭 때문입니다

2f₀ 를 받으려면 탐촉자가 f₀ 와 2f₀ 를 둘 다 담아야 합니다(A4 의 대가). 그런데 홀수 차 비선형은 스펙트럼이 송신 대역 안으로 접혀 들어옵니다. 넓은 대역폭이 아예 필요 없다는 뜻이죠. 조영제는 낮은 MI 에서도 기본파 근처에 비선형 성분을 내므로(32장), AM 이 조영 영상에서 유리합니다.

둘을 합친 AMPI(진폭+위상 동시 변조)가 상용 장비의 표준입니다 — 대신 송신이 3번이라 프레임률이 1/3 입니다.

03

움직이면 무너진다Motion breaks the cancellation

상쇄는 두 펄스가 같은 것을 본다는 전제 위에 섭니다. 그 사이에 조직이 Δx 움직이면 왕복 위상이 2kΔx 어긋나고, 선형 항이 완전히는 안 지워집니다.

인터랙티브 · 움직임이 남기는 잔류—
펄스 간격—µs
그동안 이동—µm
선형 잔류—dB
조영/조직 대비—

정지 조직은 완전히 지워집니다. 그런데 심근처럼 20~50 mm/s 로 움직이면 잔류가 −16 dB 까지 올라와 하모닉 신호를 덮어버립니다.
PRF 를 올리면 간격이 짧아져 잔류가 줍니다 — 그래서 PI 는 빠른 송신을 원합니다(→ A1).

정리하면 PI 의 대가는 둘입니다 — 송신 2배(프레임률 1/2)와 움직임 민감성입니다. 문헌에 따르면 PI 도플러는 같은 펄스의 하모닉 영상보다 조영/조직 대비를 3~10 dB 더 줍니다.

한 장 정리 · Summary
  • r(p) = a₁p + a₂p² + a₃p³ · 다항식 하나에 전부 들어 있음
  • PI: r(p) + r(−p) = 2a₂p² · 선형과 홀수까지 지워지고 짝수만 2배
  • 필터가 필요 없다 · harmonic leakage 원리적으로 0 · 대역을 좁힐 필요 없어 축방향 유지
  • AM: r(p) − 2r(p/2) = a₂p²/2 + 3a₃p³/4 · 선형만 지우고 홀수도 살림
  • 홀수는 송신 대역 안으로 접혀 들어옴 → 넓은 대역폭이 필요 없음 — 조영제에 유리 (32장)
  • AMPI = 둘을 합침 · 상용 표준 · 대신 송신 3회 → 프레임률 1/3
  • 움직임이 상쇄를 깬다 · 잔류 = 2·sin(kΔx) · PRF 5kHz 에서 5mm/s → −28dB · 20mm/s → −16dB
  • 대가 둘 · 송신 2~3배(프레임률) + 움직임 민감 → 빠른 송신을 원함 (→ A1)
