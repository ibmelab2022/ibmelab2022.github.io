부록 · Advanced Ultrasound Techniques · A6

Power Doppler and Clutter Filtering파워 도플러와 클러터 필터링

28장은 "속도를 본다"고 했습니다. 그런데 미세혈관은 속도를 잴 만큼 신호가 없습니다. 있는지 없는지만 알면 되죠. 문제는 조직이 혈액보다 1,000배 세다는 것 — 그걸 어떻게 걷어내느냐가 이 장의 전부입니다.

먼저 — 파워 도플러가 무엇인가

한 픽셀에서 펄스를 여러 번 쏘면 느린 시간 slow time 신호가 생깁니다(27·28장). 이걸 숫자 하나로 줄이는 방법이 둘입니다.

  • 컬러 도플러 · 위상 변화율로 평균 속도를 냅니다. 방향도 알지만 각도 의존(cos θ)이고 에일리어싱이 있습니다.
  • 파워 도플러 power Doppler · 위상을 버리고 진폭²을 전부 더합니다. 속도·방향은 없지만 각도 무관·에일리어싱 없음, 다 더하니 SNR 이 좋아 느리고 약한 혈류까지 민감합니다. 단, 개별 미세혈관을 분해하진 못합니다 — 회절 한계라 큰 혈관만 흐릿하게 뭉쳐 보이고, 그 한계는 A7 의 ULM 이 우회합니다.
인터랙티브 · 같은 혈관, 두 도플러 모드 ■ 접근 · ■ 후퇴
도플러 주파수—Hz
컬러 (속도 추정)—mm/s
파워 (∑진폭²)—상대
에일리어싱—

왼쪽이 컬러 도플러(속도), 오른쪽이 파워 도플러(파워) — 같은 혈관을 두 방식으로 축약한 그림입니다.
각도 θ 를 88° 로 올려 보세요 — 컬러 혈관은 색이 사그라들어 사라집니다(cos θ → 0, 드롭아웃). 그래도 파워 혈관은 그대로죠.
속도를 올리면 나이퀴스트를 넘어 컬러가 엉뚱한 방향으로 뒤집히지만(에일리어싱) 파워는 무관합니다. 그래서 파워 도플러는 "얼마나 빠른가"는 못 말해도, "혈관이 있는가"는 각도·속도에 흔들리지 않고 잡아냅니다.

01

진짜 문제는 클러터Tissue is 1,000× brighter

혈액의 산란은 적혈구의 레일리 산란(6장)이라 약한데, 주변 조직은 40~60 dB — 100~1,000배 셉니다. 클러터를 먼저 걷어내지 못하면 아무것도 못 봅니다.

그런데 조직도 움직입니다 — 그것도 혈류보다 빠르게

fd = 2v cos θ · f₀/c · 5 MHz 에서 미세혈관 1 mm/s → 6.5 Hz, 박동 조직 10 mm/s → 65 Hz. 보고 싶은 게 걷어낼 것보다 느립니다 — 주파수로는 못 가르죠.

02

전통적 방법 — 벽 필터The wall filter: high-pass on slow time

벽 필터 wall filter 는 느린 시간 신호에 고대역 통과 필터를 겁니다 — "조직은 느리고 혈류는 빠르다"는 가정이죠.

인터랙티브 · 주파수로만 자른다—
주파수 분해능—Hz
잘리는 혈류—mm/s 이하
조직 fd—Hz
판정—

파란 봉우리 = 조직, 빨간 봉우리 = 혈류, 회색 곡선이 벽 필터입니다.
조직 속도를 10 mm/s 로 올려보세요 — 조직 봉우리가 차단선을 넘어와 그대로 통과합니다 (플래시 아티팩트). 앙상블을 8~12 에 두면 필터가 뭉툭해 벽 노릇을 못 합니다 — 날카로운 벽에는 수백 장이 필요합니다.

벽 필터가 실패하는 두 지점

① 앙상블이 짧다. 기존 컬러 도플러는 한 라인당 8~12번만 쏩니다(프레임률 때문에). PRF 5 kHz·12샘플이면 주파수 분해능이 417 Hz — 50 Hz 벽을 세울 수가 없습니다. 필터가 뭉툭해서 조직도 새고 혈류도 깎입니다.

② 시간축만 본다. 픽셀 하나의 시간 신호만 보니, 조직이 빠르면 주파수가 혈류 대역으로 올라와 통과합니다 — 구별할 정보 자체가 없습니다.

03

SVD — 시공간 상관으로 자른다Spatiotemporal clutter filtering

평면파(A1)가 판을 바꿉니다. 수백~수천 장을 쌓으니 공간 × 시간을 통째로 볼 수 있죠. 프레임 전체를 한 열로 세워 쌓은 것이 Casorati 행렬 Casorati matrix — 행은 픽셀, 열은 시각입니다.

S = U Δ V*→ s(x,z,t) = Σi λi · Ui(x,z) · Vi(t)
S 를 분리 가능한 영상 Ui(x,z) 와 시간 신호 Vi(t) 의 곱으로 쪼갠 것 · λi 는 크기순 특이값 · 0 이 아닌 λi 의 개수가 곧 행렬의 계수(rank)

조직은 앞쪽 몇 개에 몰려 있으니, 그만큼 빼면 혈류만 남습니다:

sblood(x,z,t) = s(x,z,t) − Σi=1n λi Ui(x,z) Vi(t)
n = 버릴 특이벡터 수 (조직 문턱) · Demené 등의 식 (7)

그리고 남은 것의 에너지를 시간으로 적분하면 파워 도플러 영상입니다 — 이게 c1 에서 본 "진폭²을 전부 더한다"의 정확한 형태입니다:

PW(x,z) = ∫ | sblood(x,z,t) |² dt
Demené 등의 식 (8) · 위상은 버리고 에너지만 — 그래서 각도·에일리어싱과 무관
벽 필터와 SVD 는 같은 꼴 — 기저 하나만 다릅니다

벽 필터는 Sf = S E If E*(E = 푸리에 행렬), SVD 는 Sf = S V If V* — 같은 꼴입니다. 푸리에 기저는 고정, SVD 기저는 데이터가 정합니다. Eckart–Young 정리로 특이값이 더 빨리 떨어져 조직이 더 적은 차수에 압축되죠.

왜 되는가 — 조직은 연속체, 혈액은 재배열

조직이 움직이면 이웃도 같이 움직입니다 — 연속체니까요. 프레임마다의 조직 신호가 몹시 닮아 몇 개의 저차 특이벡터에 몰립니다.

혈액은 적혈구가 흐르며 재배열되니 프레임마다 스페클이 새로 그려집니다 — 닮은 구석이 없어 중간 차수에 퍼지죠. "닮음"으로 가르니 조직이 빨라도 저차에 남습니다.

판정—

조직 문턱을 0 에서 올려보세요 — 조직이 걷히며 혈관이 드러납니다. 너무 올리면 가는 혈관까지 같이 지워집니다.
조직 속도를 30 mm/s 로 올려보세요 — 벽 필터라면 이미 졌을 상황인데, 조직은 여전히 앞쪽 몇 개에 머뭅니다. 주파수가 아니라 상관으로 가르니까요.

임상 수치도 같습니다 — 소아 간은 벽 필터 차단을 120 Hz 로 올려야 했고 1.5 cm/s 이하 혈관이 사라졌습니다. 이식 신장은 90 Hz 에서 0.8 cm/s 정맥이 지워졌죠. SVD 는 둘 다 살렸고, 실제 쥐 뇌에선 앞 49개를 버려 조직을 제거했습니다. ULM(A7)은 앞(조직) + 뒤(잡음) 로 문턱을 둘 씁니다.

이 장이 A7 의 입구입니다

SVD 로 조직을 걷어내야 미세기포 하나하나가 보입니다 — 그래야 위치를 찍고 추적하죠(A7). 그리고 SVD 는 평면파 없이는 성립하지 않습니다(A1).

한 장 정리 · Summary
  • 파워 도플러 = 위상 버리고 진폭² 적분 · 각도 무관 · 에일리어싱 없음 · 속도·방향 없음
  • 클러터가 40~60 dB · 혈액은 적혈구 레일리 산란이라 약함 (6장)
  • fd = 2v cosθ·f₀/c · 미세혈관 1mm/s = 6.5 Hz < 박동 조직 10mm/s = 65 Hz
  • → 보고 싶은 게 걷어내고 싶은 것보다 느리다 — 주파수로는 못 가름
  • 벽 필터의 실패 ① · 앙상블 8~12 → 분해능 417~625 Hz — 50Hz 벽을 못 세움
  • 벽 필터의 실패 ② · 시간축만 봄 → 빠른 조직 통과 = 플래시
  • Casorati S = UΔV* · s(x,z,t) = ΣλiUi(x,z)Vi(t) · 평면파로 수백~수천 장 (A1)
  • 조직은 연속체 → 저차 · 혈액은 재배열 → 중간 차수 · 잡음은 고차 — "닮음"으로 가르니 조직이 빨라도 잡힘
  • PW(x,z) = ∫|sblood|²dt · sblood = s − Σi≤n λiUiVi (Demené 식 7·8)
  • 기저 하나 차이 · 벽필터는 고정된 푸리에, SVD 는 데이터가 정하는 적응 기저 (Eckart–Young)
  • 문턱이 둘 · 앞(조직) + 뒤(잡음) · Demené 2015 는 앞 49개
  • 실측 · 혈류 5.2 · 조직 10 mm/s 에서 벽필터 실패, SVD CNR 4.4 · 간 120Hz 차단 시 1.5cm/s 이하 소실
  • A6 이 A7 의 입구 · 조직을 걷어내야 미세기포가 보임
